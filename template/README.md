# {{theme_name}}

A modern block theme for WordPress.

## Folder Structure

- `assets/`: Static assets such as fonts and images that are not processed by build tools.
- `build/`: Compiled CSS and JS assets generated by `@wordpress/scripts`.
- `includes/`: PHP files included by `functions.php` (e.g., `enqueue.php` for enqueuing styles and scripts).
- `src/`: Source files for CSS (SCSS) and JavaScript.
- `theme.json`: WordPress block theme configuration file.

## Commands

This theme uses `@wordpress/scripts` for building and formatting assets. You can run the following commands:

- `npm start`: Starts the development build process and watches for changes.
- `npm run build`: Builds the production-ready code into the `build/` directory.
- `npm run format`: Formats both JavaScript and CSS/SCSS files to match WordPress coding standards.
- `npm run lint:js`: Lints JavaScript files using ESLint.
- `npm run lint:css`: Lints CSS/SCSS files using Stylelint.
- `npm run zip`: Packages the theme into a `.zip` file ready for installation.
